# elastic-demo-eden

Personal Elastic demo extras — scripts and configs for deploying o11y-metrics, o11y-launch-demo, and security-fc-unified content to a personal GKE cluster and Elastic serverless projects.

## Contents

- `o11y-metrics/` — OTel Astronomy Shop k8s deployment, Kibana content import, Chaos Mesh, Chatbot RAG, Snowem
- `o11y-launch-demo/` — Scenario deployer for 10 verticals against Elastic Observability serverless
- `docker/elastic-inference-proxy/` — Thin FastAPI proxy: OpenAI chat/completions → Elastic managed inference API
- `docker/snowem/` — Mock ServiceNow ITSM emulator (Node.js/SQLite)
